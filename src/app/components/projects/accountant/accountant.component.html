Accountant

Accountant es una aplicación creada en Java, Spring Boot y Thymeleaf enfocada en mantener el orden y control de las operaciones y recursos financieros de manera simple.

Registra todos y cada uno de los movimientos de dinero (efectivo y transferencias) que se realizan y presenta una serie de reportes que ayudan a la toma de decisiones financieras.

Tecnologías:

Java 100%
Spring Boot 100%
Thymeleaf 100%
